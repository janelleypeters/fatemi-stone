Slab Agent — Diagnostic

Tests each layer of the agent separately so we can see exactly which one is broken. Run them in order. Stop at the first failure and screenshot the result.

Loaded automatically if you already used the main agent in this browser. Otherwise paste it here.

Test 1 — Basic API call (no tools, no streaming)

Sends a simple "hello" message to Claude. Confirms your key works and basic networking is fine. Should complete in 1–3 seconds.

Test 2 — Streaming (no tools)

Same simple call but with stream:true. Confirms server-sent events flow from Anthropic to Safari. Should complete in 1–3 seconds.

Test 3 — Web search tool

Asks Claude to do exactly one web search for "Crema Marfil marble slab" with the web_search server tool. Confirms web_search is enabled on your account and reachable. Should complete in 10–30 seconds.

Test 4 — Web search tool with structured output

Exactly what the main agent's search stage does, but simplified. Combines web_search with a structured output tool. Should complete in 30–60 seconds.

If Test 1 fails → API key or network issue.
If Test 1 passes but Test 2 fails → streaming/SSE issue in Safari.
If Tests 1 & 2 pass but Test 3 fails → web_search tool isn't enabled on your account or is misconfigured.
If Tests 1–3 pass but Test 4 fails → the main agent's prompt or tool combination is the problem.
